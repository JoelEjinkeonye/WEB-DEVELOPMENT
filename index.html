HTML Cheat Sheet

HTML Cheat Sheet

HTML Tags

  • <html> - Root element
  • <head> - Metadata container
  • <body> - Content container

HTML Attributes

  • class - Defines one or more classes for an element
  • id - Defines a unique id for an element

HTML Properties

  • innerText - Sets or returns the text content of a node and its descendants
  • innerHTML - Sets or returns the HTML content of an element

Example Code Snippets

<div class="container">
    <h1>Hello, World!</h1>
</div>
CSS Cheat Sheet

CSS Cheat Sheet

CSS Selectors

  • element - Selects all elements of a specific type
  • .class - Selects all elements with a specific class
  • #id - Selects a single element with a specific ID
  • selector1, selector2 - Selects all elements that match either selector1 or selector2
  • selector1 selector2 - Selects all selector2 elements that are descendants of selector1
  • selector1 > selector2 - Selects all selector2 elements that are direct children of selector1

CSS Properties

  • color - Sets the color of text
  • font-size - Sets the size of the font
  • background-color - Sets the background color of an element
  • margin - Sets the margin around an element
  • padding - Sets the padding inside an element
  • border - Sets the border around an element
  • width - Sets the width of an element
  • height - Sets the height of an element

Box Model

The CSS box model describes the content and spacing of an element. It consists of:

  • content - The actual content of the element
  • padding - The space between the content and the border
  • border - The border surrounding the padding
  • margin - The space outside the border

Example Code Snippets

/* CSS Comment */
.selector {
    color: #333; /* Text color */
    font-size: 16px; /* Font size */
    background-color: #f0f0f0; /* Background color */
    margin: 10px; /* Margin */
    padding: 20px; /* Padding */
    border: 1px solid #ccc; /* Border */
    width: 200px; /* Width */
    height: 100px; /* Height */
}
JavaScript Cheat Sheet

JavaScript Cheat Sheet

JavaScript Variables

  • var - Declares a variable (ES5)
  • let - Declares a block-scoped variable (ES6)
  • const - Declares a block-scoped constant (ES6)

JavaScript Data Types

  • String - A sequence of characters
  • Number - Numeric values
  • Boolean - Logical value, either true or false
  • Array - Ordered collection of values
  • Object - Collection of key-value pairs

JavaScript Functions

  • function functionName() { ... } - Declares a named function
  • () => { ... } - Arrow function (ES6)
  • functionName = function() { ... } - Function expression
  • return - Returns a value from a function

Example Code Snippets

// JavaScript Comment
var x = 5; // Declaring and assigning a variable
let y = "Hello"; // Declaring and assigning a variable with let
const z = [1, 2, 3]; // Declaring and assigning a constant array
function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("World")); // Output: Hello, World!
